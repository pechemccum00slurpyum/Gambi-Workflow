Workflow Diagram

        

            
                flowchart TD
                A[Receive Inquiry from CS] --> B[Check User Account for Multiple IPs, Usernames, or High-Risk Email]
                
                %% High-Risk Email Check
                B --> X[Check High-Risk Emails at APIVoid: https://www.apivoid.com/tools/email-reputation-check/]
                X --> C[Assess Account for Multiple IPs, Usernames, or High-Risk Email]
                
                %% Suspicious Activity Detected and Temporary Suspension
                C --> D{Suspicious Activity Detected?}
                D -->|Yes| S[Temporarily Suspend User Account / All Related Accounts]
                S --> Y[Check if User Has Uploaded ID Document]
                
                %% ID Document Verification Workflow
                Y --> Z{Does ID Match with Other Users?}
                
                Z -->|Yes| I[Ban All Accounts with Matching ID Permanently]
                I --> V[Leave Remark: Confirmed Multi Abuser / Related Accounts Have Been Perma Banned, No Further Action Is Needed]
                
                Z -->|No| F[Request KYC Verification: Selfie with ID + Gambi Paper]
                F --> U[Leave Remark: Multi / Promocode / Suspended / Selfie+ID+Gambi Requested]
                
                %% Request and Verification of KYC
                U --> G{Received KYC Submissions?}
                
                G -->|Yes| H[Check if User IDs Match Across Accounts]
                G -->|No| M[Inform User of Ineligibility Due to Missing KYC]
                
                %% ID Verification Workflow for KYC Submissions
                H -->|Yes, Matching IDs| I
                H -->|No, IDs Do Not Match| J[Update User KYC Level to KYC 3]
                
                %% KYC Level Update Workflow
                J --> K{Is KYC1 Information Complete?}
                K -->|No| L[Correct User FirstName and LastName in Admin Panel per Document]
                K -->|Yes| M[Proceed with Bonus Process]
                
                %% No Suspicious Activity Workflow
                D -->|No| M[Proceed with Bonus Process]
                
                %% Bonus Process Workflow
                M --> N[Verify Promotion or Referral Code]
                N --> O[Check User Eligibility]
                O --> P{User Eligible for Bonus?}
                
                P -->|Yes| Q[Add Bonus to User Account]
                P -->|No| R[Inform User of Ineligibility]
            
                %% Finalize and Close Inquiry
                Q --> T[Close Inquiry]
                R --> T
                V --> T
